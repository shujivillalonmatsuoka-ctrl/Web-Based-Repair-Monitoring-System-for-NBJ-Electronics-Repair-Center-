# Web-Based-Repair-Monitoring-System-for-NBJ-Electronics-Repair-Center-
The system will allow customers and administrators to access repair monitoring services through a web browser. Customers can log in to view repair status, while administrators can manage repair records, customer accounts, and updates.

Project Name:
NBJ Web-Based Repair Monitoring System

Developer:
@shujimatsuoka

Overview:
A web system that allows customers to monitor repair status online
and allows admins to manage repair transactions.

How to Run:
1. Open Eclipse
2. Import Project
3. Connect MySQL Database
4. Run on Apache Tomcat Server
